# Contributing To MLdeck Public Documentation

This repository is for public MLdeck documentation only. Contributions should improve clarity, safety, accuracy, and discoverability without exposing private implementation details or sensitive data.

## Public Documentation Only

Do not contribute private MLdeck source code, runtime logic, worker logic, model internals, export internals, proprietary prompts, admin logic, Firebase/server configuration, credentials, generated packages, private reports, or private screenshots.

## Do Not Upload Sensitive Data

Do not upload datasets with personal, financial, health, customer, or sensitive business data. Do not include private CSVs, row-level examples from real users, internal identifiers, or confidential business context.

## Do Not Upload Generated Export Packages

Do not upload generated export packages unless manually approved by the repository owner. This includes ONNX files, PKL files, wheel files, ZIP packages, runtime bundles, manifests tied to private data, or generated validation artifacts.

## Do Not Upload Secrets Or Private Files

Do not upload `.env` files, service-account files, API keys, tokens, certificates, private keys, credentials, internal configuration, or private reports.

## Safe Contribution Types

Safe contributions include:

- Typo fixes.
- Documentation improvements.
- Non-sensitive public examples.
- Link corrections.
- Responsible-use wording improvements.
- Clarifications that strengthen privacy, validation, export, and safety boundaries.

## Security Reporting

Do not open public issues with sensitive security details, private datasets, secrets, credentials, customer data, or exploit steps that expose private information.

Use a private contact route on https://mldeck.com/ if available.

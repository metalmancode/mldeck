# Security

Do not submit sensitive datasets through GitHub issues.

Do not post CSVs with personal data, private business data, health data, financial data, credentials, secrets, or internal identifiers.

Do not post API keys, tokens, `.env` files, private exports, service account files, certificates, private keys, or credentials.

Security concerns should be reported privately through the contact route on https://mldeck.com/ if available.

Public issues should be limited to documentation feedback and non-sensitive examples.

# Sample Reports

Sample reports may be added later only after manual review.

Do not add:

- Private user data.
- Real customer data.
- Sensitive datasets.
- Secrets, credentials, tokens, or environment values.
- Misleading claims about maturity or validation.
- Unreviewed export packages.
- CSV, ONNX, PKL, wheel, archive, or generated package files unless manually approved later.

Any future sample report should be clearly labeled as documentation material and should avoid claims that exceed the evidence shown in the report.
